R Functions for the Standard Normal Distribution

There are four R functions for analysis of the normal distribution

  1. dnorm(): compute normal pdf
  2. rnorm(): simulate random numbers from normal distribution
  3. pnorm(): compute normal CDF
  4. qnorm(): compute normal quantiles (inverse CDF)

To see the arguments of an R function use the args() function:

# arguments of dnorm() function
args(dnorm)
## function (x, mean = 0, sd = 1, log = FALSE) 
## NULL

Notice by default the dnorm() function set the mean equal to zero and the standard deviation to 1, which gives the standard normal pdf.

Computing Probabilities from the Standard Normal Distribution

The CDF of the standard normal distribution is \(F_{X}(x) = \Phi (x)\). Probabilities of the form \(\Pr(x_{1} \leq X \leq x_{2})\) are computed as \(\Phi(x_{2}) - \Phi(x_{1})\). Consider computing \(\Pr(-1 \leq X \leq 2)\). The R calculations are:

# compute Pr(-1 <= X <= 2) where X ~ N(0,1)
pnorm(2) - pnorm(1)
## [1] 0.136

Properties of the Standard Normal Distribution

# Pr(-1 < X < 1)
pnorm(1) - pnorm(-1)
## [1] 0.683
# Pr(-2 < X < 2)
pnorm(2) - pnorm(-2)
## [1] 0.954
# Pr(-3 < X < 3)
pnorm(3) - pnorm(-3)
## [1] 0.997

Computing Quantiles from the Standard Normal Distribution

The inverse CDF (quantile function) of the standard normal distribution is used to find quantiles of the distribution.

# Compute 1%, 2.5% and 5% quantiles of X ~ N(0,1)
qnorm(c(0.01, 0.025, 0.05))
## [1] -2.33 -1.96 -1.64
# Compute 99%, 97.5% and 95% quantiles of X ~ N(0,1)
qnorm(c(0.99, 0.975, 0.95))
## [1] 2.33 1.96 1.64
# Compute median
qnorm(0.5)
## [1] 0

Value-at-Risk Calculations for Simple Returns

Let \(R\) denote the simple monthly return on Microsoft stock and assume that \(R\sim N(0.05,(0.10)^{2}\). Consider an initial investment of \(W_{0}=\$10,000\). To compute the 1% and 5% VaR values over the next month use:

mu.R = 0.05
sd.R = 0.10
w0 = 10000
q.01.R = mu.R + sd.R*qnorm(0.01)
q.05.R = mu.R + sd.R*qnorm(0.05)
VaR.01 = abs(q.01.R*w0)
VaR.05 = abs(q.05.R*w0)
VaR.01
## [1] 1826
VaR.05
## [1] 1145

Hence with 1% and 5% probability the loss over the next month is at least $1,826 and $1,145, respectively.

Value-at-Risk calculations for cc Returns

Let \(r\) denote the continuously compounded return on Microsoft stock and assume that \(r\sim N(0.05,(0.10)^{2})\). To compute the 1% and 5% VaR values over the next month use:

mu.r = 0.05
sd.r = 0.10
q.01.R = exp(mu.r + sd.r*qnorm(0.01)) - 1
q.05.R = exp(mu.r + sd.r*qnorm(0.05)) - 1
VaR.01 = abs(q.01.R*w0)
VaR.05 = abs(q.05.R*w0)
VaR.01
## [1] 1669
VaR.05
## [1] 1082

Notice that when \(1+R=e^{r}\) has a log-normal distribution, the 1% and 5% VaR values (losses) are slightly smaller than when \(R\) is normally distributed. This is due to the positive skewness of the log-normal distribution.
